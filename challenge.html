Final Challenge

Hello my dear friends, you have reached my final javascript challenge.

Bellow you will find the rules to this exercise.

Objective: Create a dynamic webpage that shows a list of books.

Must haves

  • Rendering a complete list (like a table) getting the content on your webserver
  • Add possibility to use the existing filters: Author, title and publisher
  • Loader when the table is loading

Nice to have

  • Add pagination (5 items per page)
  • Use some effects to smooth the user experience
  • Make it prettier
